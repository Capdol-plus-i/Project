## 브랜치 전략 (Branch Strategy)
Git Flow를 기반으로 하며, 다음과 같은 브랜치를 사용합니다.

- **Main Branch**
  - 배포 가능한 상태의 코드를 유지

- **{name} Branch**
  - 팀원 각자의 개발 브랜치
  - 모든 기능 개발은 이 브랜치에서 이루어짐
